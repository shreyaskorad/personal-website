Coaching Leadership Through Role-Playing Games

April 18, 2026 · 2 min read · modern-l-d, work, strategy

A learning strategy is only credible when it can explain how capability survives contact with live work. The better test is simple: does the design change the next decision in real work?

Leaders care about capability building, but many L&D functions still operate as order-takers instead of business-shaping partners.

The mechanism is not a cleaner strategy slide. It is whether managers and workflows move with the strategy.

Borrow one method from recent evidence, run it in one team next week, and keep it only if behavior changes in your workflow.

The source points to an operating-model problem, not a messaging problem: strategy works only when managers, workflows, and reinforcement all move with it.

The source is really describing a shift in operating model. The argument works only if strategy changes what gets reinforced in live work, not just what gets documented on slides.

Across learning science, behavior science, organizational design, management science, the same pattern shows up. The intervention matters only if it changes behavior around a live decision.

Baldwin & Ford (1988) matters here because canonical frame for transfer and maintenance of learned capability.

Grossman & Salas (2011) matters here because practical evidence on support, follow-up, and opportunity to perform.

The tradeoff is coherence versus reality: elegant strategy language can hide the fact that no one has designed for reinforcement, feedback, or opportunity to perform.

The broader literature also warns that results depend heavily on context, incentive mix, and implementation details rather than the mechanic alone.

Start with one target behavior in one live context. Then rebuild the learning strategy outward from that moment.

Michie, van Stralen, & West (2011) matters here because keeps design anchored to target behavior in context.

The useful test is not whether the intervention looks engaging. It is whether the next decision gets sharper.
