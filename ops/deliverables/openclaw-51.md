## Prioritization Rules

**Garden Task Priority Matrix**:
```
┌─────────────────────┬─────────────────────┬─────────────────────┐
│                    │ CRITICAL            │ IMPORTANT          │
│                    │ (Will Worsen)       │ (Will Degrade)      │
│                    │                     │                     │
│ EMERGENCY           │ Q1: Do Immediately  │ Q2: Do Within 24h   │
│ (Immediate Threat)  │                     │                     │
│                    │                     │                     │
├─────────────────────┼─────────────────────┼─────────────────────┤
│                    │ IMPORTANT          │ ROUTINE             │
│                    │ (Maintenance)       │ (Future Planning)   │
│ ROUTINE            │                     │                     │
│ (Regular Tasks)    │ Q3: Schedule        │ Q4: Time permitting  │
│                    │                     │                     │
│                    │                     │                     │
└─────────────────────┴─────────────────────┴─────────────────────┘

Priority Categories:
- Q1 (Do Immediately): Emergency care - dying plants, pest infestations, disease outbreaks
- Q2 (Do Within 24h): High priority maintenance - water stress, pruning needs, minor disease
- Q3 (Schedule): Regular maintenance - fertilizing, weeding, minor pruning
- Q4 (Time Permitting): Low priority - aesthetic improvements, future planning
```

**Emergency vs. Routine Prioritization**:
- **Emergency**: Immediate action required (plants dying, severe pests, disease spreading)
- **Critical**: Must address within 24 hours (water stress, minor disease, severe infestation)
- **Important**: Address within 1 week (pruning, fertilizing, minor pest control)
- **Routine**: Address monthly/seasonally (weeding, mulching, general maintenance)
- **Future Planning**: Address when time permits (improvements, expansion, redesign)

**Time-Based Prioritization**:
- Morning tasks (high energy): Heavy lifting, pruning, planting
- Mid-day tasks (moderate energy): Watering, weeding, light maintenance
- Afternoon tasks (lower energy): Observation, planning, light cleanup
- Evening tasks (relaxed): Final checks, record-keeping, reflection

## Review Cadence

**Daily**: End of each garden session
- Update daily task completion status
- Record plant observations and issues
- Track time invested
- Identify any new tasks created

**Weekly**: Friday
- Review entire garden health and catch-up progress
- Update all tracking dashboards
- Identify priorities for next week
- Schedule any specialized care (professional services, suppliers)
- Plan upcoming maintenance schedule

**Monthly**: Last day of each month
- Comprehensive garden assessment
- Evaluate catch-up progress against goals
- Adjust maintenance schedule based on seasons and conditions
- Review plant health and identify necessary changes
- Plan for upcoming season's needs
- Assess equipment and supplies inventory

**Seasonally**: Beginning of each season (Spring, Summer, Autumn, Winter)
- Seasonal garden planning and preparation
- Inventory assessment and ordering
- Equipment maintenance and preparation
- Plant selection for upcoming season
- Adjust care schedule for seasonal changes

**Annually**: End of year
- Complete garden health review
- Evaluate catch-up progress for the year
- Document lessons learned
- Plan improvements for next year
- Update plant inventory
- Review equipment needs
